# B0 브랜드 가이드 스킬

## 개요

이 스킬은 B0 (비제로) 프로젝트의 브랜드 일관성을 유지하기 위한 가이드를 제공합니다.

## 언제 사용하나요?

- 새로운 UI 컴포넌트나 페이지를 디자인할 때
- 사용자에게 보여지는 문구나 메시지를 작성할 때
- 에러 메시지, 안내 문구 등을 작성할 때
- 색상, 폰트, 스타일을 결정할 때
- 세계관 관련 용어를 사용할 때

## 핵심 원칙

### 브랜드 성격

- **신비롭고 몽환적인**: 지하 0층, 비행선, 이세계
- **따뜻하고 포근한**: 게스트하우스, 힐링
- **낭만적이고 감성적인**: 여행, 별멍, 불멍
- **차분하고 사려 깊은**: 자기성찰, 일기

### 톤앤매너

- 사용자를 **"여행자님"**으로 호칭
- 명령조 대신 **제안/초대** 형식
- **따뜻하고 친근한** 어조
- 과한 이모지 사용 자제

### 좋은 예시

```
"안녕하세요, 여행자님. B0에 오신 것을 환영합니다."
"오늘은 어느 도시로 떠나볼까요?"
"잠시 쉬어가는 건 어떨까요?"
```

### 피해야 할 예시

```
"지금 바로 클릭하세요!" (강압적)
"굿굿~ 짱이에요!!" (과한 표현)
"Error: Invalid input" (차갑고 기술적)
```

## 상세 가이드

상세한 브랜드 가이드는 `reference.md` 파일을 참조하세요. 다음 내용이 포함되어 있습니다:

1. 브랜드 아이덴티티
2. 세계관 (6개 도시, 공간 설명)
3. 디자인 시스템 (색상, 타이포그래피, 컴포넌트)
4. 톤앤매너 상세
5. 이미지 가이드
6. 접근성 가이드
7. 반응형 디자인
8. 용어 사전
