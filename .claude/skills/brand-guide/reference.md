# B0 (비제로) 브랜드 가이드

이 문서는 B0 프로젝트의 디자인, 문서 작성, 마케팅 등 모든 제작물에서 참고할 브랜드 가이드입니다.

---

## 1. 브랜드 아이덴티티

### 프로젝트명

- **정식 명칭**: B0 (비제로)
- **부제**: 지하 0층
- **태그라인**: "당신 마음이 잠시 쉬어갈 곳"
- **영문**: B0 - Floor Zero

### 브랜드 핵심 가치

| 가치     | 설명                |
|--------|-------------------|
| **탈출** | 반복되는 일상에서 벗어나는 경험 |
| **연결** | 진정성 있는 대화와 관계 형성  |
| **성찰** | 자기 자신을 돌아보는 시간    |
| **힐링** | 마음의 안식과 회복        |
| **모험** | 새로운 세계로의 여정       |

### 브랜드 성격

- **신비로운 (Mysterious)**: 숨겨진 지하 0층, 이세계로 통하는 비밀스러운 공간
- **따뜻한 (Warm)**: 게스트하우스의 아늑함, 여행자들 간의 포근한 교류
- **낭만적 (Romantic)**: 비행선 여행, 노을 지는 하늘, 판타지 도시들
- **차분한 (Calm)**: 자기성찰과 내면 탐색에 적합한 고요한 분위기
- **환상적 (Fantastical)**: 현실과 다른 세계, 스팀펑크와 판타지가 어우러진 비주얼

## 2. 세계관

### 핵심 컨셉

**"어느 날 발견한 신비한 앱, B0. 존재하지 않는 층에서 출발하는 비행선을 타고 이세계 도시를 여행하며 자신을 돌아보고 사람들을 만나는 곳."**

- **B0 터미널**: 지하 0층에 숨겨진 비행선 정류장. 모든 여정의 시작점
- **비행선**: 도시와 도시를 연결하는 낭만적인 이동 수단 (스팀펑크 스타일)
- **게스트하우스**: 각 도시에서 여행자들이 머무는 공간. 대화와 성찰의 장소
- **여행자**: 서비스 사용자들의 정체성. "사용자"가 아닌 "여행자"로 호칭

### 도시 테마

| 도시명      | 테마 | 설명              | 분위기       |
|----------|----|-----------------|-----------|
| **세렌시아** | 관계 | 인간관계와 소통에 대한 성찰 | 노을빛 항구 마을 |
| **로렌시아** | 회복 | 지친 마음의 휴식과 치유   | 숲 속 오두막   |
| **엠마시아** | 희망 | 빛이 머무는 초원의 도시   | 밝은 초원     |
| **다마린**  | 고요 | 물안개 속 항구 도시     | 안개 낀 항구   |
| **갈리시아** | 성찰 | 석양이 머무는 순례의 길   | 황금빛 길     |

### 주요 공간

| 공간            | 용도            | 분위기 키워드   |
|---------------|---------------|-----------|
| **B0 터미널**    | 도시 선택, 비행선 탑승 | 신비로운, 기대감 |
| **비행선**       | 이동 중 대기 화면    | 낭만적, 설렘   |
| **게스트하우스 거실** | 메인 허브, 공간 선택  | 아늑한, 따뜻한  |
| **사랑방**       | 단체 채팅 공간      | 활기찬, 편안한  |
| **라운지**       | 1:1 대화 공간     | 조용한, 친밀한  |
| **개인 숙소**     | 일기/문답지 작성     | 고요한, 성찰적  |

## 4. UI 컴포넌트 가이드

### 버튼

#### Primary 버튼 (주요 액션)

- **배경**: 브랜드 퍼플
- **텍스트**: 화이트
- **모서리**: 12px (둥근)
- **호버**: 약간 어두운 퍼플
- **용도**: 회원가입, 예매하기, 제출 등 주요 행동

#### Secondary 버튼 (보조 액션)

- **배경**: 투명
- **테두리**: 라이트 퍼플 (50% 불투명도)
- **텍스트**: 라이트 퍼플
- **모서리**: 12px
- **호버**: 배경에 퍼플 틴트
- **용도**: 취소, 둘러보기, 건너뛰기 등

#### Ghost 버튼 (텍스트 링크)

- **배경**: 없음
- **텍스트**: 라이트 퍼플
- **호버**: 텍스트 밝아짐
- **용도**: 자세히 보기, 더 알아보기 등

### 카드

- **배경**: 글래스모피즘 적용 (반투명 카드 네이비)
- **테두리**: 흰색 10% 불투명도
- **모서리**: 16px
- **패딩**: 내부 여백 24px
- **호버**: 배경 불투명도 증가, 테두리 밝아짐

### 입력 필드

- **배경**: 반투명 카드 네이비
- **테두리**: 흰색 20% 불투명도
- **텍스트**: 화이트
- **Placeholder**: 그레이 500
- **모서리**: 12px
- **포커스 시**: 테두리가 브랜드 퍼플로 변경

### 체크박스

- **기본**: 흰색 30% 불투명도 테두리
- **체크 시**: 브랜드 퍼플 배경 + 흰색 체크 아이콘

### 아이콘

- **스타일**: 선형 (outline) 아이콘
- **라이브러리**: Lucide Icons 권장
- **크기**: 16px (작음), 20px (기본), 24px (큼)
- **색상**: 주변 텍스트와 동일하게

## 5. 톤 앤 매너 (글쓰기)

### 기본 원칙

| 원칙         | 설명                   | 예시                  |
|------------|----------------------|---------------------|
| **따뜻하게**   | 사용자를 "여행자"로 대하며 친근하게 | "환영합니다, 여행자님"       |
| **부드럽게**   | 명령조보다 권유형 사용         | "떠나볼까요?" vs "떠나세요"  |
| **상상력 자극** | 세계관에 맞는 은유적 표현       | "비행선을 타고" vs "이동하기" |
| **간결하게**   | 핵심 메시지만 전달           | 짧고 임팩트 있는 문장        |
| **존중하며**   | 정중하되 딱딱하지 않게         | 존댓말 + 부드러운 어미       |

### 예시 문구

#### 좋은 예 ✅

- "잠깐 떠나볼까요"
- "반복되는 일상, B0에서 비행선을 타고 여행자들을 만나보세요."
- "당신만의 이야기를 시작하세요"
- "쉼터에서 여행자들과 소소한 대화, 저녁 9시에는 호스트가 준비한 특별한 시간도 있어요"
- "스스로에게 질문하고 답하며 내면의 목소리에 귀 기울이세요"
- "가장 기억에 남는 순간은.."
- "더 많은 도시가 곧 열립니다"
- "사전 예약자에게는 오픈 알림과 특별한 환영 선물을 드려요"

#### 피해야 할 예 ❌

- "지금 바로 가입하세요!" → 너무 강압적
- "채팅방에 입장합니다" → 세계관 무시 (게스트하우스로)
- "사용자님" → 여행자로 호칭
- "기능을 사용해 보세요" → 딱딱한 표현
- "로딩 중..." → 세계관에 맞지 않음
- "에러가 발생했습니다" → 딱딱함

### 에러 메시지

| 상황     | 일반적 표현         | B0 스타일 표현                     |
|--------|----------------|-------------------------------|
| 로딩     | 로딩 중...        | 비행선이 출발 준비 중이에요               |
| 404    | 페이지를 찾을 수 없습니다 | 앗, 길을 잃었어요. 터미널로 돌아갈까요?       |
| 서버 에러  | 서버 오류          | 비행선에 잠시 문제가 생겼어요. 곧 다시 출발할게요  |
| 네트워크   | 연결 실패          | 구름이 너무 두꺼워요. 잠시 후 다시 시도해 주세요  |
| 입력 오류  | 올바른 이메일을 입력하세요 | 탑승권을 받을 주소를 확인해 주세요           |
| 포인트 부족 | 포인트가 부족합니다     | 여행 자금이 부족해요. 일기를 쓰거나 대화해 볼까요? |
| 빈 상태   | 데이터가 없습니다      | 아직 여행 기록이 없어요. 첫 여정을 시작해 볼까요? |

## 6. 이미지 가이드

### 이미지 스타일

| 요소         | 스타일                 | 설명          |
|------------|---------------------|-------------|
| **전체 분위기** | 판타지 + 스팀펑크          | 신비롭고 낭만적인 톤 |
| **색감**     | 보라-핑크 노을, 골드 하이라이트  | 따뜻하면서 몽환적   |
| **조명**     | 황혼, 골든아워, 은은한 실내 조명 | 부드럽고 따뜻한 빛  |
| **품질**     | 고퀄리티 CG, 디테일한 텍스처   | 게임/영화급 배경   |

### 공간별 배경 이미지

| 공간         | 이미지 설명                        | 분위기     |
|------------|-------------------------------|---------|
| **히어로 섹션** | 구름 위를 나는 거대한 비행선, 보라-핑크 노을 하늘 | 웅장함, 설렘 |
| **도시 선택**  | 바다 위 판타지 섬들, 산과 숲이 있는 지도 스타일  | 모험, 호기심 |
| **게스트하우스** | 나무 인테리어, 따뜻한 조명, 창밖 풍경        | 아늑함, 안식 |
| **개인 숙소**  | 작은 1인실, 책상과 창문, 부드러운 조명       | 고요함, 성찰 |
| **사전 예약**  | 이미지 없이 깊은 남색 배경               | 집중, 신뢰  |

### 이미지 사용 원칙

- **오버레이**: 배경 이미지 위에 어두운 그라데이션 오버레이를 적용해 텍스트 가독성 확보
- **일관성**: 모든 이미지는 보라-핑크-골드 색조로 통일감 유지
- **품질**: 저해상도 이미지 사용 금지, 최소 1920x1080 이상

### 아이콘 및 일러스트

- **스타일**: 세계관과 어울리는 선형 아이콘 또는 미니 일러스트
- **색상**: 보라-화이트 계열, 필요시 골드 포인트
- **예시**:
    - 도시 마커: 커스텀 위치 핀
    - 비행선: 비행선 실루엣 아이콘
    - 대화: 말풍선 아이콘
    - 일기: 노트/책 아이콘

## 7. 접근성

### 필수 고려사항

- **색상 대비**: 텍스트와 배경의 대비율 4.5:1 이상 유지
- **키보드 접근**: 모든 버튼과 링크는 키보드로 접근 가능해야 함
- **포커스 표시**: 현재 선택된 요소가 명확히 보여야 함 (보라색 테두리)
- **대체 텍스트**: 모든 이미지에 설명 텍스트 제공

### 텍스트 가독성

- **최소 크기**: 본문 텍스트 14px 이상
- **줄 간격**: 본문 텍스트는 160% 줄 간격 권장
- **문단 너비**: 한 줄에 35-40자 (한글 기준) 권장

## 10. 용어 사전

| 서비스 용어       | 일반 용어       | 설명                   |
|--------------|-------------|----------------------|
| **B0 (비제로)** | 앱/서비스       | 지하 0층, 모든 여정의 시작점    |
| **여행자**      | 사용자/회원      | B0을 이용하는 모든 사람       |
| **비행선**      | 이동 수단       | 도시 간 이동 시 탑승하는 운송체   |
| **터미널**      | 홈/메인 화면     | B0의 비행선 정류장          |
| **도시**       | 카테고리/테마     | 각기 다른 주제를 가진 여행 목적지  |
| **게스트하우스**   | 채팅방/커뮤니티    | 여행자들이 머무는 공간         |
| **사랑방**      | 단체 채팅       | 게스트하우스 내 단체 대화 공간    |
| **라운지**      | 1:1 채팅      | 게스트하우스 내 개인 대화 공간    |
| **개인 숙소**    | 마이페이지/개인 공간 | 일기, 문답지 작성 공간        |
| **대화 카드**    | 대화 주제       | 대화를 유도하는 질문 카드       |
| **문답지**      | 자기성찰 질문     | 도시별 성찰 질문지           |
| **포인트**      | 재화/크레딧      | 서비스 내 활동 보상 및 결제 수단  |
| **티켓**       | 이용권         | 비행선 탑승에 필요한 권한       |
| **체크인**      | 입장          | 게스트하우스에 도착하여 입실      |
| **체크아웃**     | 퇴장          | 게스트하우스에서 퇴실하여 B0로 복귀 |
| **연장**       | 기간 연장       | 게스트하우스 체류 기간 추가      |
| **호스트**      | AI/운영자      | 대화를 이끌고 이벤트를 진행하는 존재 |
