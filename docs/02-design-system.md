## 2. 타이포그래피

```css

## 4. 모서리 (Border Radius)

| 크기 | 값 | 용도 |
|------|-----|------|
| **sm** | 6px | 태그, 뱃지 |
| **md** | 8px | 입력 필드 |
| **lg** | 12px | 버튼, 작은 카드 |
| **xl** | 16px | 카드, 모달, 팝업 |
| **full** | 9999px | 원형 버튼, 아바타 |

**원칙**: B0은 부드럽고 따뜻한 느낌을 위해 **둥근 모서리**를 기본으로 사용합니다.

## 5. 컴포넌트 스펙

### 5.1 버튼 (Button)

#### Primary 버튼
```
배경: #9333EA (Brand Purple)
텍스트: #FFFFFF (White)
모서리: 12px
높이: 44px (터치 최소 크기)
패딩: 16px 24px
호버: 약간 어두운 퍼플 (#7C3AED)
```

#### Secondary 버튼
```
배경: transparent
테두리: #C084FC 50% opacity, 1px
텍스트: #C084FC (Light Purple)
모서리: 12px
높이: 44px
호버: 배경에 퍼플 틴트 (rgba(192, 132, 252, 0.1))
```

#### Ghost 버튼
```
배경: transparent
텍스트: #C084FC (Light Purple)
호버: 텍스트 밝아짐
```

### 5.2 카드 (Card) - 글래스모피즘

```
배경: rgba(26, 27, 60, 0.7) + backdrop-blur(12px)
테두리: rgba(255, 255, 255, 0.1), 1px
모서리: 16px
패딩: 16px
호버: 배경 불투명도 증가 (0.85), 테두리 밝아짐
```

### 5.3 입력 필드 (Input)

```

### 5.4 채팅 메시지 버블

## 7. 그라데이션

### 7.1 노을 그라데이션 (Hero)

```css
background: linear-gradient(135deg, #9333EA 0%, #F97316 50%, #F59E0B 100%);
```
용도: 히어로 이미지, 특별 배경

### 7.2 배경 그라데이션

```css
background: linear-gradient(180deg, #1A1B3C 0%, #0F0F23 100%);
```
용도: 섹션 배경, 페이지 전환

### 7.3 오버레이 그라데이션 (이미지 위)

```css
background: linear-gradient(180deg, transparent 0%, rgba(15, 15, 35, 0.9) 100%);
```
용도: 이미지 위 텍스트 가독성

## 9. 접근성

### 9.1 색상 대비

- 텍스트와 배경의 대비율: **4.5:1 이상**
- 주요 색상 조합:
  - White (#FFFFFF) on Deep Navy (#0F0F23): ✅ 15.5:1
  - White (#FFFFFF) on Card Navy (#1A1B3C): ✅ 12.8:1
  - White (#FFFFFF) on Brand Purple (#9333EA): ✅ 4.6:1

### 9.2 터치 영역

- 최소 터치 영역: 44px x 44px
- 버튼 높이: 44px 이상

### 9.3 포커스 표시

- 포커스 시 테두리: Brand Purple (#9333EA)
- 키보드 접근: 모든 버튼과 링크
