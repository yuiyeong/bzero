사용 방법

  • 각 링크를 클릭하면 해당 화면들의 와이어프레임을 볼 수 있습니다
  • 화면은 iPhone X (375 x 812) 크기로 렌더링됩니다
  • 브라우저에서 스크린샷을 찍어 Figma에 임포트할 수 있습니다
  • Chrome DevTools > More tools > Screenshot 기능 사용 권장

인증 플로우 (2단계)

Step 1: Supabase Auth
회원가입 → 이메일/비밀번호 입력 → Supabase에서 JWT 발급

Step 2: Backend Onboarding
프로필 설정 → 닉네임/이모지 입력 → Backend에서 User 생성 + 1000P 지급
